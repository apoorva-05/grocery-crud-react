A simple React application that helps users manage their daily grocery items efficiently.

Features : 
   - Add new items.
   - Mark completed items.
   - Edit and delete items.
   - Clear all feature to delete all items.
   - Toast notifications.
   - Local storage to save data.


The project is built using React functional components and hooks.
It also uses libraries such as React Toastify, NanoID, and React Icons.
This project demonstrates CRUD operations and state management in React.
